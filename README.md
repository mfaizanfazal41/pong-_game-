# Pong Game 🎮

A classic Pong game built with **HTML, CSS, and JavaScript**.  
This project recreates the retro arcade experience directly in your browser.

---

## 🚀 How to Play
- **Left Paddle**: Use `W` (up) and `S` (down) keys  
- **Right Paddle**: Use `Arrow Up` and `Arrow Down` keys  
- Ball bounces off paddles and walls.  
- If the ball goes out of bounds, it resets to the center.

## ▶️ Run the Game
1. Clone or download this repository.  
2. Open `index.html` in any modern browser.  
3. Play Pong instantly — no installation required!
